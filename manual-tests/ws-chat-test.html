Эта страница больше не используется. Проект переведен на Spring Security + JAAS + HTTP Basic, поэтому ручная проверка REST и WebSocket выполняется через backend-страницу:

http://localhost:8433/rest-endpoints-smoke.html

Открой ее через backend, пройди HTTP Basic-аутентификацию и используй встроенный клиент. Текущий пользователь определяется через /me, а WebSocket подключается под тем же аутентифицированным principal.
